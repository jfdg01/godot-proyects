# Flappy Bird - Godot Learning Project

This project is for **learning Godot 4 and GDScript**.

## Rules

- **Do not modify code directly.** When the user asks how to do something, explain the concept and the steps — let the user apply the changes themselves.
- **Be didactic.** Explain *why* things work, not just *what* to write. Build understanding, not just solutions.
- **Use Godot terminology** correctly (nodes, scenes, signals, resources, etc.).
